Instructions

            
Quick guide for controllers — pick a format, fill fields, generate, copy.

          

        

        


        
    
          
  • 
            1) Pick a PDC Type
            Use the sidebar: ATC24 Standard or VATUSA.
          
    • 
          
  • 
            2) Fill the fields
            
    
              Callsign
              Route
              (ATC24) Dest / Climb / RWY / Dep Freq
              (VATUSA) Dep/Dest / Equip / Alt / SID
            
    
          
    • 
          
  • 
            3) Generate
            Click Generate. Squawk uses 0–7 only and won’t repeat in the same tab session.
          
    • 
          
  • 
            4) Copy & send
            Click Copy Output. It wraps output in backticks for Discord.
          
    • 
          
  • 
            VATUSA ground contact toggle
            PUSH → CTC ___ FOR PUSH • TAXI → CTC ___ FOR TAXI
          
    • 
          
  • 
            Theme
            Toggle Light/Dark in the sidebar. Your choice is saved.
          
    •
